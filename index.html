Calm Flow

A subtle pink-ish noise generator that responds to typing cadence with no UI elements.

Signal Path

White Noise Loop → Pink Tilt Filter → Lowpass Filter → Master Gain → (Optional) Breath LFO → Visibility Gain → Output

Parameter Mapping

Fast typing (80ms IKI): Lower gain (0.18), darker sound (800Hz), slower breathing (0.07Hz)

Slow typing (500ms IKI): Higher gain (0.28), brighter sound (2200Hz), faster breathing (0.12Hz)

Autoplay Policy

Audio starts automatically on page load with no user gesture required. The engine begins playback immediately when the page opens.

Safety Features

Master gain capped at ≤ 0.3, all parameters have safety bounds, graceful Web Audio API fallback, idempotent start/stop operations.
